#  Simple BloomFilter

## Prerequisites
- `Java 21+`

## Getting Started

![demo](demo.gif)

```shell
shell:>help

AVAILABLE COMMANDS
Cli Command
       add: 
       check: 

shell:>add hello_world
Added hello_world

shell:>check hello_world
Might contain 'hello_world'

shell:>check helloworld
Definitely not contain 'helloworld'
```
